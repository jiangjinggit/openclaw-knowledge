# 👤 USER.md - 用户画像与偏好

> 告诉 AI "你是谁、你的偏好、你的上下文"

---

## 📋 基本信息

### 必填项

| 字段 | 说明 | 示例 |
|------|------|------|
| **Name** | 你的名字或昵称 | 张三、Alex |
| **How to call** | AI 怎么称呼你 | 老板、主人、张三 |
| **Timezone** | 你的时区 | Your/Timezone (UTC+8) |
| **Identity** | 你的身份 | 独立开发者、创业者、学生 |

### 填写示例

```markdown
- Name: 张三
- How to call: 老板
- Timezone: Your/Timezone (UTC+8)
- Identity: 独立开发者
```

## 🧭 决策偏好

描述你的决策偏好，让 AI 知道你怎么做决定。

### 通用模板

```markdown
- 优先"可验证进展"，避免只停留在讨论
- 优先"能沉淀为系统"的方案（SOP、模板、自动化）
- 任务必须具备 owner、deadline、acceptance
- 结论必须附证据，风险必须可执行地管理
```

### 自定义示例

```markdown
- 优先"快速验证"，不追求完美
- 优先"低成本试错"，不怕失败
- 优先"用户反馈"，不凭感觉
```

## 🎯 使用说明

### 必改项
1. **基本信息**：填写你的真实信息
2. **核心路线**：描述你的战略
3. **重点业务方向**：列出你的优先级

### 可选项
1. **关注的结果**：根据你的目标调整
2. **决策偏好**：根据你的风格调整
3. **工作时间**：根据你的作息调整

### 修改建议

> 💡 **重要**：越具体越好，AI 才能更好地理解你。

**第1步**：复制一个示例配置
**第2步**：修改成你的信息
**第3步**：测试效果
