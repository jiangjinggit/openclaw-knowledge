## ⚡ 3步上手

### Step 1：安装 OpenClaw

```bash
npm install -g openclaw
openclaw gateway start
```

### Step 2：复制模板

从本知识库复制你需要的模板，优先改这3个：
- `USER`
- `IDENTITY`
- `AGENTS`

### Step 3：跑第一个任务

示例：
- 用 `07_TASK_任务卡模板` 建立任务
- 用 `08_RESULT_结果卡模板` 做回传
- 发布前跑 `09_CHECKLIST_门禁检查清单`
