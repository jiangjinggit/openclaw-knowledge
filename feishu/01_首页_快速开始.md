## ⚡ 3步上手

### Step 1: 安装 OpenClaw（5分钟）

```bash
# 安装
npm install -g openclaw

# 启动
openclaw gateway start
```

### Step 2: 下载配置模板（2分钟）

点击下方按钮下载配置包，或者直接复制本知识库的配置文件。

📦 **下载配置包**
- [GitHub 下载](链接待补充)
- [网盘下载](链接待补充)

### Step 3: 修改配置（10分钟）

只需要修改 3 个文件：
1. **USER.md** - 填写你的信息（名字、时区、业务）
2. **IDENTITY.md** - 给 AI 起个名字
3. **AGENTS.md** - 选择需要的 AI 员工

---

## 📚 配置文件说明

| 文件 | 作用 | 必改 |

## ❓ 常见问题

<details>
<summary>Q: 我不会编程，能用吗？</summary>

能。只需要修改配置文件（纯文本），不需要写代码。
</details>

<details>
<summary>Q: 需要多少钱？</summary>

OpenClaw 免费，只需支付 AI 模型 API 费用（Claude/GPT，约 $0.01/次）。
</details>

<details>
<summary>Q: 数据安全吗？</summary>

完全安全。所有数据都在你的电脑上，不会上传到任何服务器。
</details>

<details>
<summary>Q: 可以自定义 AI 员工吗？</summary>

可以。你可以删除不需要的角色，也可以添加新角色。
</details>
