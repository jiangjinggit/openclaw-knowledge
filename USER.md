## 使用说明

这个文件告诉 AI "你是谁、你的偏好、你的上下文"。

**修改建议：**
1. 填写你的基本信息（名字、时区、身份）
2. 描述你的核心战略（你在做什么）
3. 列出你最关注的结果（你想要什么）
4. 说明你的决策偏好（你怎么做决定）
5. 列出你的重点业务（你的优先级）

**记住：**
- 越具体越好，AI 才能更好地理解你
- 可以随时更新，根据你的变化调整
- 不要写太多，保持简洁

**示例场景：**

如果你是独立开发者：
```
Name: 张三
Identity: 独立开发者
Timezone: Asia/Shanghai (UTC+8)

核心路线：产品 + 内容双飞轮
重点业务：
1. SaaS 产品开发
2. 技术博客
3. 开源项目

决策偏好：
- 优先可验证进展
- 优先能沉淀为系统的方案
```

如果你是内容创作者：
