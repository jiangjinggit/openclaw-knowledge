# OpenClaw Knowledge 销售栈

## 目标
明确 `openclaw-knowledge` 的产品不是单点售卖，而是一套从免费内容到高客单升级的销售栈。

---

## 第一层：免费内容层
### 作用
- 吸引流量
- 建立信任
- 解释方法论

### 载体
- GitHub README
- 公众号深度文章
- 小红书避坑内容
- 免费清单 / 路线图 / 导航页

### 输出目标
- 让用户知道你有真实模板和方法论
- 让用户愿意领取 lead magnet

---

## 第二层：Lead Magnet 层
### 作用
- 收联系方式
- 识别高意向用户
- 为低价产品成交做准备

### 载体
- 多 Agent 配置自查清单
- 任务卡 + 结果卡 + 门禁清单
- 新手上手路线图
- AI 公司模板 5 件套简版

### 输出目标
- 从“看内容”变成“留下联系方式”

---

## 第三层：低价成交层
### 作用
- 拿第一笔钱
- 验证真实付费意愿
- 让用户从内容消费者变成产品用户

### 核心产品
- 多 Agent 协作模板包
- AI 公司配置模板包
- 飞书知识库模板包
- 多 Agent 协作 SOP 包

### 输出目标
- 建立基础现金流
- 识别最强成交产品

---

## 第四层：中价升级层
### 作用
- 提高客单价
- 让用户买的不只是模板，而是系统化结构

### 载体
- 更完整的方法论包
- 飞书交付版
- 训练营资料包
- 工作坊配套材料

### 输出目标
- 让低价产品用户升级到更深的交付关系

---

## 第五层：高价培训 / 顾问层
### 作用
- 承接团队客户
- 提高收入密度
- 形成更高客单的交付路径

### 载体
- 训练营
- 工作坊
- 企业内训
- 顾问交付前置资料包

### 输出目标
- 拉高单客价值
- 打开企业和团队场景

---

## 推荐成交路径
公开内容
→ 免费清单 / 路线图
→ 多 Agent 协作模板包
→ AI 公司配置模板包 / 飞书模板包
→ 训练营 / 企业内训 / 顾问升级

---

## 关键原则
1. 免费内容不承担全部价值。
2. lead magnet 负责筛出高意向用户。
3. 低价产品负责成交验证。
4. 中高价产品负责拉高客单和关系深度。
5. 所有内容都要服务于销售栈中的某一层。
