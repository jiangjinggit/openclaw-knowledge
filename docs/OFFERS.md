# OpenClaw Knowledge 可售卖 Offers 设计

## 目标
把 `openclaw-knowledge` 从“内容与模板仓库”进一步推进为：

> **可以直接承接免费引流、低价成交、培训升级和顾问扩展的知识产品体系**

---

## Offer 1：多 Agent 协作模板包
### 适合谁
- 想快速搭建第一套 OpenClaw 协作结构的人
- 看过内容后想直接照着搭的人

### 交付物
- 核心配置 5 件套
- 角色模板
- 任务卡 / 结果卡 / 门禁清单
- 最小可用示例

### 建议价格
`39-99 元`

### 作用
- 最容易成交的低价入口产品

---

## Offer 2：AI 公司配置模板包

## 推荐成交路径
免费内容 / lead magnet
→ 多 Agent 协作模板包
→ AI 公司配置模板包 / SOP 包
→ 飞书知识库模板包
→ 训练营 / 工作坊
→ 企业内训 / 顾问交付

---

## 判断原则
每一个 offer 都应该满足至少两项：
1. 能直接节省用户搭建时间
2. 能从现有知识资产中低成本打包出来
3. 能和公众号/小红书内容自然联动
4. 能为更高价产品做前置成交
