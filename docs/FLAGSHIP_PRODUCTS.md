# OpenClaw Knowledge 旗舰产品建议

## 目标
从已经规划的一批模板包、SOP、培训资料里，进一步收窄出**最值得优先卖的 3 个旗舰产品**，避免产品线过早发散。

---

## 旗舰产品 1：多 Agent 协作模板包
### 为什么优先级最高
- 最贴当前仓库已有资产
- 打包成本最低
- 最适合做第一笔低价成交
- 最容易通过 GitHub / 公众号 / 小红书承接

### 适合谁
- 第一次搭 OpenClaw 多 Agent 协作的人
- 想直接拿现成结构的人

### 建议价格
`39-99 元`

### 作用
- 作为最低门槛的入门成交产品
- 承担第一层付费验证功能

---

## 旗舰产品 2：AI 公司配置模板包
### 为什么应该第二个卖
- 比基础模板包更有方法论价值
- 更像你自己的产品化表达
- 更容易建立独特认知

### 适合谁
- 想把个人协作升级成更完整工作流的人
- 想做内容工厂 / 产品工厂 / 研究工厂的人

### 建议价格
`99-199 元`

### 作用
- 拉高客单价
- 建立“DJY 方法论模板”心智

---

## 旗舰产品 3：飞书知识库交付模板包
### 为什么值得做
- 差异化更明显
- 更适合培训、顾问、企业交付场景
- 更容易承接高意向与高客单用户

### 适合谁
- 顾问 / 培训交付方
- 团队协作和企业试点用户

### 建议价格
`99-299 元`

### 作用
- 把模板产品升级成交付型产品
- 连接企业内训和顾问升级路径

---

## 当前不建议优先卖的
### 训练营 / 工作坊资料包
原因：
- 客单更高，但前置信任要求也更高
- 需要先有前面三类产品做成交和验证

### 过多的细分模板同时上线
原因：
- 会稀释仓库主卖点
- 反而不利于快速建立认知

---

## 推荐顺序
1. 多 Agent 协作模板包
2. AI 公司配置模板包
3. 飞书知识库交付模板包
4. 再考虑训练营 / 工作坊升级

---

## 判断原则
未来如果再新增产品，优先和这 3 个旗舰产品比较：
- 是否更容易成交？
- 是否更贴当前仓库资产？
- 是否更利于后续培训升级？

如果都没有明显优势，就不该优先上线。
