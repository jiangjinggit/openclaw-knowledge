# OpenClaw Knowledge 资产地图

## 目标
把仓库内容从“文件列表”变成“资产视图”，便于后续内容复用、产品包装和培训交付。

---

## 一、核心配置资产
用于搭建 OpenClaw 多 Agent 协作的最小骨架。

- `SOUL.md`
- `IDENTITY.md`
- `USER.md`
- `AGENTS.md`
- `MEMORY.md`

### 价值
- 个人用户：快速起步
- 顾问/培训：作为第一版演示模板
- 内容创作：作为“标准配置模板”讲解基础

---

## 使用建议
今后仓库新增文件时，优先给自己回答两个问题：
1. 这个内容属于哪类资产？
2. 它对应免费引流、低价产品、还是培训升级？

如果两者都答不清，说明它不该直接加入主仓库。
