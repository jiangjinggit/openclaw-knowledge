# OpenClaw Knowledge 内容产品计划

## 目标
把 `openclaw-knowledge` 从知识仓库继续推进成：

> **可持续产出低价知识产品、模板包、训练营材料和顾问前置资产的内容产品底座**

---

## 一、优先级最高的内容产品

### 1. 多 Agent 协作模板包
**定位**：给第一次搭 OpenClaw 多 Agent 协作的人一个可直接复用的模板包。

**建议包含**：
- 核心配置 5 件套
- 任务卡/结果卡模板
- 协作规则模板
- 最小可用示例

**适合谁**：
- 独立开发者
- 想搭 AI 公司工作流的人
- 培训/咨询客户

**建议价格**：`39-99 元`

---

### 2. AI 公司配置模板包
**定位**：把“多 Agent 协作”从抽象概念变成更具体的公司式协作结构。

**建议包含**：
- CEO / PM / Build / Research / Growth / Ops 配置模板
- 角色分工说明
- 常见任务流示例
- 验收与门禁模板

**建议价格**：`99-199 元`

---

### 3. 飞书知识库交付模板包
**定位**：给培训、顾问、团队交付场景使用的飞书版本模板。

**建议包含**：
- 知识库目录结构
- 首页文案
- 各模块拆分文档
- 交付说明

**建议价格**：`99-299 元`

---

### 4. 多 Agent 协作 SOP 包
**定位**：强调“流程可复用”，而不仅仅是“配置文件可复用”。

**建议包含**：
- 任务创建 SOP
- 结果回传 SOP
- 质量门禁 SOP
- 内容工厂/产品工厂的流程示例

**建议价格**：`99-299 元`

---

## 二、适合后续升级的内容产品

### 5. 训练营讲义包
- 面向训练营、课程或小型工作坊
- 可作为高客单培训升级材料

### 6. 企业内训模板包
- 面向团队协作、角色设计、交付规范
- 可作为顾问与培训升级路径

### 7. 顾问交付前置资料包
- 面向接单、交付、企业 PoC
- 提升顾问服务标准化程度

---

## 三、建议的发布顺序
1. 多 Agent 协作模板包
2. AI 公司配置模板包
3. 飞书知识库交付模板包
4. 多 Agent 协作 SOP 包
5. 训练营讲义包
6. 企业内训模板包

---

## 四、与内容平台的联动方式
### 公众号
- 发方法论长文
- CTA 指向模板包 / 飞书模板 / SOP 包

### 小红书
- 发避坑与入门内容
- CTA 指向免费清单 / 基础模板包

### GitHub
- 承接免费版与认知入口
- 提供基础模板与方法论导航

### 私域 / 社群
- 承接训练营、工作坊、顾问升级

---

## 五、判断标准
未来新增一个产品前，优先回答：
1. 它是否能独立成交？
2. 它是否明显节省用户时间？
3. 它是否能从现有知识资产直接打包出来？
4. 它是否能作为训练营或咨询的前置产品？

若至少满足 3 项，则值得推进。
